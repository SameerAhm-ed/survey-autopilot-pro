# 🚀 AutoSurvey Pro by Sameer Ahmed

A smart Chrome Extension that automates survey completion with human-like behavior.

## ✨ Features

- ⚡ Auto-completes surveys instantly
- 🧠 Smart answering (MCQ + text)
- 🎯 Multiple modes: Fast, Safe, Stealth
- 🔀 Randomized answers to avoid detection
- 🧾 Handles one-question-per-page surveys
- 🔐 Basic license system

## 🛠️ Tech Stack

- JavaScript (Vanilla)
- Chrome Extension API (Manifest v3)
- DOM Automation
